Question 8 (**)#

A pressure gauge at an elevation 4.8 m on the side of a storage tank containing oil reads 34.7 kPa. Another gauge at elevation 2.2 m reads 57.5 kPa. Calculate the specific weight, density and specific gravity of the oil. {8.77 kN/m3; 894 kg/m3; 0.894}

The first step is calculate the pressure and height difference between the two points and then use the equations for the requested parameters

\[ \Delta p= p_{gauge2}-p_{gauge1} \]
\[ \Delta h= h_1-h_2 \]

The next step is calculate the specific weight, density and specific gravity of the oil with the following equations:

\[ \gamma=\frac {\Delta p}{\Delta h} \]
\[ \rho=\frac {\gamma}{g} \]
\[ s= \frac {\gamma}{\gamma_{water}} \]

The units of the variables are: \(\gamma= [N/m^3], p= [Pa], h= [m], g = [m/s^2], \rho =[kg/m^3]\)

Solution

Show code cell outputs Hide code cell outputs
\[\begin{split} \begin{aligned} \Delta_{p} &= p_{gauge_{2}} - p_{gauge_{1}} \\&= 57.50 - 34.70 \\&= 22.80 \; \;\textrm{(kPa)}\\[10pt] \\[10pt] \Delta_{h} &= h_{1} - h_{2} \\&= 4.80 - 2.20 \\&= 2.60 \; \;\textrm{(m)}\\[10pt] \\[10pt] \gamma &= \frac{ \Delta_{p} }{ \Delta_{h} } \\&= \frac{ 22.80 }{ 2.60 } \\&= 8.77 \; \;\textrm{(kN/m3)}\\[10pt] \\[10pt] \gamma_{oil} &= 8769.23 \; \;\textrm{(N/m3)} \\[10pt] \rho_{oil} &= \frac{ \gamma_{oil} }{ g } \\&= \frac{ 8769.23 }{ 9.81 } \\&= 893.91 \; \;\textrm{(kg/m3)}\\[10pt] \\[10pt] s_{oil} &= \frac{ \rho_{oil} }{ \rho_{water} } \\&= \frac{ 893.91 }{ 1000 } \\&= 0.89 \\[10pt] \end{aligned} \end{split}\]
